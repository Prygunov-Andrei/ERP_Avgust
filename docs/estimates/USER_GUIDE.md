# Сметы — Руководство пользователя

## Импорт сметы

### Импорт из Excel

1. Откройте смету и нажмите **Импорт файла**
2. Нажмите **Excel** и выберите файл (.xlsx)
3. В окне предпросмотра проверьте распознанные строки
4. При необходимости назначьте строки заголовками разделов (кнопка папки слева от строки)
5. Нажмите **Импортировать** — строки будут добавлены в смету

### Импорт из PDF (постраничный)

Для больших PDF-файлов (десятки и сотни страниц) используется постраничная обработка через ИИ. Это может занимать **10 и более минут**.

1. Откройте смету и нажмите **Импорт файла**
2. Нажмите **PDF** и выберите файл
3. Начнётся постраничная обработка — строки появляются в таблице по мере распознавания
4. Вы можете работать с уже загруженными строками, не дожидаясь окончания

### Сворачивание диалога во время обработки

Если нужно заняться другими делами пока PDF обрабатывается:

1. Нажмите кнопку **Свернуть** (или нажмите Esc / кликните вне окна)
2. Диалог свернётся в компактный **синий индикатор** в правом нижнем углу экрана
3. Индикатор показывает текущий прогресс (номер страницы и процент)
4. Можно работать с другими разделами ERP — индикатор не мешает
5. Когда обработка завершится — **прозвучит звуковой сигнал** и диалог автоматически развернётся на экране предпросмотра

**Отмена импорта из свёрнутого вида:** нажмите **X** на индикаторе.

**Кнопка "Остановить"** — прекращает обработку оставшихся страниц, но уже распознанные строки сохраняются и доступны для импорта.

> **Важно:** Не уходите со страницы сметы пока идёт обработка. Переход на другую страницу ERP прервёт отслеживание прогресса (хотя обработка на сервере продолжится).

## Работа с разделами

Разделы помогают структурировать смету (например: «Вентиляция», «Электрика», «Водоснабжение»).

### Назначить строку заголовком раздела

1. Найдите строку, которая должна стать заголовком раздела
2. Нажмите кнопку с иконкой папки слева от строки
3. Строка станет заголовком раздела:
   - Выделена синим цветом и жирным шрифтом
   - Все данные кроме наименования исчезают
   - Все строки ниже до следующего заголовка автоматически принадлежат этому разделу

### Снять раздел

1. Нажмите кнопку папки на заголовке раздела (синяя строка)
2. Раздел будет снят — строка вернётся к обычному виду (с нулевыми ценами)
3. Все строки этого раздела переместятся в предыдущий раздел

## Редактирование строк

- **Inline-редактирование**: кликните на ячейку, чтобы изменить значение
- **Изменение ширины колонок**: перетащите границу заголовка колонки
- **Удаление строки**: кнопка корзины справа от строки
- **Массовое удаление**: отметьте строки чекбоксами слева → нажмите кнопку «Удалить (N)»

---

## Подбор цен и работ

- **Подобрать цены** — автоматически ищет соответствия в каталоге товаров и подставляет цены материалов
- **Подобрать работы** — автоматически ищет соответствия в каталоге работ и подставляет цены работ

---

## Добавление строк

- **Кнопка «Строка»** — добавить одну строку вручную
- **Вставить из Excel** — скопируйте строки из Excel и вставьте через диалог
- **Импорт файла** — загрузите Excel или PDF файл целиком
