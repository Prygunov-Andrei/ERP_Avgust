# Руководство пользователя: Настраиваемые столбцы сметы

## Обзор

Каждая смета имеет настраиваемую конфигурацию столбцов. Вы можете:
- Скрывать/показывать стандартные столбцы
- Добавлять пользовательские столбцы (числа, текст, даты, чекбоксы, выпадающие списки)
- Создавать вычисляемые столбцы с формулами
- Сохранять и загружать шаблоны конфигурации
- Экспортировать смету в Excel с учётом настройки столбцов

## Как открыть настройки столбцов

1. Откройте смету
2. Перейдите на вкладку «Строки»
3. Нажмите кнопку с шестерёнкой (⚙) в правом верхнем углу панели инструментов

## Настройка столбцов

### Скрыть/показать столбец

В левой панели списка столбцов:
- Каждый столбец отображает иконку «глаз» если он скрыт
- Выберите столбец → снимите/поставьте галку «Видимый» в правой панели

Стандартные столбцы (номер, наименование, цены, итоги) нельзя удалить, но можно скрыть.

### Изменить порядок столбцов

Используйте кнопки ↑↓ рядом с каждым столбцом в левой панели.

### Добавить пользовательский столбец

В верхней части левой панели нажмите одну из кнопок:

| Кнопка | Описание | Пример использования |
|--------|----------|---------------------|
| **+ Число** | Числовое поле | «Наценка %», «Скидка» |
| **+ Текст** | Текстовое поле | «Примечание», «Марка» |
| **+ Дата** | Поле даты | «Дата поставки» |
| **+ Выбор** | Выпадающий список | «Статус» (Заказано / В пути / Получено) |
| **+ Чекбокс** | Галочка да/нет | «Согласовано», «Оплачено» |
| **+ Формула** | Вычисляемый столбец | «Итого с наценкой» |

### Настройка свойств столбца

Выберите столбец в списке, в правой панели доступны:

- **Название** — отображаемый заголовок
- **Ключ (key)** — уникальное имя для использования в формулах (только латиница, цифры, `_`)
- **Ширина** — ширина столбца в пикселях
- **Десятичные знаки** — для чисел и формул (2 = «1 234.56»)
- **Видимый** — показывать/скрывать
- **Редактируемый** — разрешить ввод (формулы всегда нередактируемы)
- **Итоги** — показывать сумму в подвале таблицы

### Удалить столбец

Нажмите 🗑 рядом со столбцом в списке. Стандартные (встроенные) столбцы удалить нельзя.

При удалении столбца данные в строках сметы сохраняются. Если столбец с тем же ключом будет создан снова, данные подхватятся автоматически.

## Формулы

### Синтаксис

Формулы записываются в поле «Формула» при выборе столбца типа «Формула».

Поддерживаются:
- Арифметика: `+`, `-`, `*`, `/`
- Скобки: `(выражение)`
- Числа: `100`, `1.5`
- Ссылки на столбцы: по `key` (например, `quantity`, `material_unit_price`)
- Функции: `round()`, `max()`, `min()`, `abs()`

### Примеры

| Формула | Описание |
|---------|----------|
| `quantity * material_unit_price` | Итого материалы |
| `line_total * 1.2` | С наценкой 20% |
| `line_total * (1 + markup_pct / 100)` | С динамической наценкой |
| `round(quantity * price, 2)` | Округление до 2 знаков |
| `max(material_total, work_total)` | Максимум из двух столбцов |

### Доступные переменные

В формулах можно использовать:
- Все стандартные поля: `item_number`, `quantity`, `material_unit_price`, `work_unit_price`, `material_total`, `work_total`, `line_total`
- Все пользовательские числовые столбцы (по их `key`)
- Другие формульные столбцы (вычисляются в правильном порядке)

Список доступных переменных показывается под полем формулы в настройках.

### Ошибки в формулах

Если формула содержит ошибку (неизвестная переменная, синтаксическая ошибка), она подсвечивается красным под полем ввода. Сохранить конфигурацию с ошибками всё равно можно, но столбец с ошибкой покажет «—» вместо значения.

Циклические зависимости (A ссылается на B, B ссылается на A) блокируют сохранение.

## Шаблоны столбцов

### Сохранить шаблон

1. Настройте столбцы как нужно
2. Нажмите «Сохранить как шаблон»
3. Введите название и описание
4. Нажмите «Сохранить»

### Загрузить шаблон

В нижней части диалога настройки столбцов отображаются сохранённые шаблоны. Нажмите на название шаблона, чтобы применить его к текущей смете.

### Удалить шаблон

Нажмите 🗑 рядом с шаблоном. Удаление шаблона не влияет на сметы, к которым он уже был применён.

### Сбросить к стандартным

Кнопка «Сбросить» возвращает конфигурацию к 10 стандартным столбцам.

## Экспорт в Excel

1. Откройте смету
2. Нажмите «Экспорт в Excel» (иконка Download) в панели действий

Файл Excel включает:
- Заголовок с номером и названием сметы
- Только видимые столбцы (скрытые не экспортируются)
- Разделы как заголовки-строки
- Вычисленные формулы
- Строку итогов для столбцов с включенным суммированием

## FAQ

**Что происходит с данными при удалении столбца?**
Данные остаются в базе, но не отображаются. Если создать столбец с тем же ключом, данные восстановятся.

**Влияет ли настройка столбцов одной сметы на другие?**
Нет, конфигурация хранится per-estimate. Для копирования используйте шаблоны.

**Вставка из Excel работает с кастомными столбцами?**
Нет, вставка из Excel работает только со стандартными 6 полями (Наименование, Модель, Ед.изм., Кол-во, Цена мат., Цена раб.). Кастомные столбцы заполняются вручную.

**Копируются ли столбцы при создании новой версии сметы?**
Да, `column_config` и `custom_data` всех строк копируются в новую версию.
