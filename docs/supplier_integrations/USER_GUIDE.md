# Руководство пользователя: Интеграции поставщиков

## Что это такое

Система интеграции с поставщиками позволяет:
- Импортировать полный каталог товаров поставщика (названия, описания, фото, ТХ)
- Получать закупочные цены и рекомендованные цены
- Отслеживать наличие товаров по складам поставщика
- Выбирать поставщика при автоподборе цен в сметах

## Где найти товары поставщика

Все товары поставщиков доступны в **Каталог → Номенклатура**. Отдельной страницы «Каталог поставщика» нет — всё в одном месте.

### Фильтры в Номенклатуре

- **Поставщик** — показать товары от конкретного поставщика (например, БРИЗ)
- **В наличии** — показать только товары с остатками на складах
- **Поиск** — по названию, артикулу

В таблице видны:
- Миниатюра товара
- Бренд
- Категория
- Статус

## Карточка товара

Нажмите на товар, чтобы открыть полную информацию.

### Вкладка «Информация»

- **Галерея изображений** — горизонтальная лента миниатюр, клик для увеличения
- **Бренд / Серия**
- **Описание** — текстовый блок
- **Технические характеристики** — таблица ключ-значение
- **Документация** — ссылки на буклеты и инструкции в PDF

### Вкладка «Поставщики»

Таблица всех предложений поставщиков для данного товара:

| Поставщик | НС-код | Артикул | Закупочная | РИЦ | Наличие | Обновлено |
|-----------|--------|---------|-----------|-----|---------|-----------|
| БРИЗ      | НС-0082588 | ZES 500x250-12 | 21 949₽ | 32 923₽ | 6 шт | 13.03.2026 |

Нажмите на строку, чтобы раскрыть детали по складам (название склада, количество).

### Вкладка «Алиасы»

Альтернативные названия товара для улучшения поиска и сопоставления.

### Вкладка «Цены»

История цен из оплаченных счетов (кто, когда, за сколько поставлял).

## Подключение поставщика

1. Перейдите в **Настройки → Интеграции поставщиков**
2. Нажмите **Добавить поставщика**
3. Заполните форму:
   - **Название** — произвольное имя (например, «Бриз»)
   - **Провайдер** — выберите тип API
   - **URL API** — адрес API поставщика
   - **Authorization Header** — ключ авторизации (получите у поставщика)
4. Нажмите **Добавить**

## Синхронизация каталога

После добавления поставщика:

1. На карточке поставщика нажмите **Синхр. каталог**
2. Процесс выполняется в фоне — можете продолжать работу
3. Когда синхронизация завершится, вы увидите статус и статистику

Что происходит при синхронизации каталога:
- Загружаются все категории, бренды и товары из API поставщика
- Для каждого товара создаётся своя карточка в Номенклатуре (1:1)
- Карточки обогащаются фотографиями, описаниями и характеристиками

## Синхронизация остатков и цен

1. Нажмите **Синхр. остатки** на карточке поставщика
2. Обновятся закупочные цены и наличие по складам

## Автоподбор цен в смете с выбором поставщиков

При работе со сметой:

1. Откройте смету → нажмите **Автоподбор**
2. В диалоге настройте:
   - **Поставщики** — выберите, от каких поставщиков искать цены (по умолчанию — все)
   - **Стратегия** — «Самая дешёвая цена» или «Последняя из счетов»
3. Нажмите **Найти цены**
4. Просмотрите результаты:
   - 📦 — цена из каталога поставщика
   - 📄 — цена из оплаченного счёта
   - Раскройте строку для выбора альтернативного предложения
5. Отметьте нужные строки и нажмите **Применить**

## Маппинг категорий

На странице деталей интеграции → вкладка **Маппинг категорий**:
- Каждой категории поставщика можно сопоставить нашу категорию

## Логи синхронизаций

На вкладке **Логи** доступна история всех синхронизаций:
- Дата, тип (каталог/остатки), статус
- Количество обработанных, созданных, обновлённых товаров
- Количество ошибок и их детали
- Длительность выполнения
